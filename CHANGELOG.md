## 0.0.2

* Added drag-and-drop support with collision handling (snap-back).
* Added resize support from both bottom-right and top-left corners.
* Added optional grid lines visualization (`showGrid` parameter).
* Optimized `DashboardController` to use `Map` for O(1) item access.
* Added minimum size constraints (1x1) during resizing.
